# README.md

## Cloning & Building

### Cloning

```bash
git clone --branch master --recurse-submodules --shallow-submodules https://github.com/xexaaron/AbyssEngine.git
```

### Building

```bash
cmake -S ./AbyssEngine -B ./AbyssEngine/build
cmake --build ./AbyssEngine/build
```

## Versions

- CMake 3.28.3
- C++ 23
- SPIR-V 1.6
- Vulkan 1.3.250
